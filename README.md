## ✨ Features

- **100% Native Swift** — Zero Electron. Zero Chromium. Unmatched battery efficiency.
- **120Hz ProMotion Optimized** — Flawlessly tied to the display refresh rate.
- **Syntax Highlighting** — Auto-detects code across **30+ languages**.
- **Pin & Expand Previews** — Hover to peek. Click to pin with fluid spring animations.
- **Drag-to-Resize** — Grab any edge or corner. Panels remember your size.
- **Extreme Privacy** — Zero analytics. AES-256 encrypted. 100% offline.
- **Liquid Glass UI** — Deep blur matrices, dynamic mesh backgrounds.

## 🚀 Installation

### Option 1: Direct Download
Download the latest `Pasty.dmg` from the [Releases Page](https://github.com/JordanH22/pasty/releases/latest).

### Option 2: Homebrew (Recommended)
```bash
brew install --cask pasty
```
